1. 下载代码 `git clone git@github.com:jrg-team/draw-a-rose.git`
2. 文件夹内运行 `parcel src/index.html`
3. 浏览器中查看 `http://localhost:1234`
4. 预览链接：[点击预览](https://jrg-team.github.io/draw-a-rose/)
5. 效果图：
